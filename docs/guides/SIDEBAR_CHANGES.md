# 사이드 메뉴 변경 사항

## ✅ 적용된 변경 사항

### **사이드 메뉴에 "즉시 송출" 항목 추가**

**위치**: `src/components/layout/Sidebar.tsx`

**변경 내용**:
- "전송·스케줄" 섹션에 "즉시 송출" 메뉴 항목 추가
- 아이콘: `Radio` (lucide-react)
- 경로: `/send/broadcast`

**메뉴 구조**:
```
전송·스케줄
  ├─ 전송 설정 (/send/setup)
  ├─ 즉시 송출 (/send/broadcast) ← 새로 추가
  └─ 스케줄 관리 (/send/schedule)
```

---

## 🎯 사용 방법

### **사이드 메뉴에서 즉시 송출 접근**

1. **사이드 메뉴 열기**
   - 왼쪽 사이드바에서 "전송·스케줄" 클릭
   - 하위 메뉴 확장

2. **"즉시 송출" 클릭**
   - 스케줄 관리 페이지로 이동
   - 음원 목록에서 원하는 음원 선택 후 "송출" 버튼 클릭

3. **송출 다이얼로그에서 즉시 송출 실행**
   - "즉시 송출" 선택
   - 채널 선택
   - "송출" 버튼 클릭

---

## 📝 참고 사항

- **즉시 송출 페이지**는 현재 **스케줄 관리 페이지**를 재사용합니다
- 사이드 메뉴에서 접근하는 것 외에는 동일한 기능을 제공합니다
- 향후 별도의 "즉시 송출" 전용 페이지로 분리할 수 있습니다
