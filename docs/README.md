# SmartTTS 프로젝트 문서

이 폴더에는 프로젝트 관련 문서들이 종류별로 정리되어 있습니다.

## 📁 폴더 구조

### `/deployment` - 배포 가이드
- `DEPLOYMENT_GUIDE.md` - 전체 배포 가이드
- `HOSTING_SETUP_GUIDE.md` - 호스팅 서버 설정 가이드
- `QUICK_START_TABLET.md` - 태블릿 빠른 시작 가이드
- `TABLET_SETUP_GUIDE.md` - 태블릿 설정 가이드

### `/api` - API 관련 문서
- `/broadcast` - 방송 API 관련 문서들
- `README.md` - Broadcasting API 문서

### `/guides` - 개발 가이드
- `IMPLEMENTATION_PLAN.md` - 구현 계획
- `NEXT_STEPS.md` - 다음 단계
- `PC_BROADCAST_ARCHITECTURE.md` - PC 방송 아키텍처
- `PC_BROADCAST_SOLUTION.md` - PC 방송 솔루션
- `TABLET_BROADCAST_SCENARIO.md` - 태블릿 방송 시나리오
- `TABLET_PLAYER_IMPLEMENTATION.md` - 태블릿 플레이어 구현
- `FRONTEND_CHANGES_SUMMARY.md` - 프론트엔드 변경 요약
- `SIDEBAR_CHANGES.md` - 사이드바 변경사항

### `/troubleshooting` - 문제 해결 가이드
- 각종 FIX, GUIDE, CHECKLIST 문서들
- 문제 발생 시 참고할 수 있는 문서들

## 📝 주요 문서

### 배포
- Vercel 배포: 루트의 `vercel.json` 참고
- 호스팅 배포: `deployment/HOSTING_SETUP_GUIDE.md` 참고

### API
- Vercel 서버리스 함수: `api/broadcast-node/index.js`
- 호스팅 PHP API: `server/php/api/broadcast/index.php`
